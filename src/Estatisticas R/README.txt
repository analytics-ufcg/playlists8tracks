-O script "match_10porcento_melhores.playlists.R" recebe como entrada 2 arquivos: um contendo estatisticas de
similaridade do timbre para as playlists 10% mais e menos populares e um arquivo contendo os dados para todas as musicas obtidas.
Em seguida ele faz um match desses arquivos de modo a obter um unico arquivo contendo os dados das musicas das playlists mais e menos populares.


-O script "calcula_media_mediana_variancia_sd_para_atributos_das_musicas_das_playlists.R" calcula estatisticas para as musicas e agrupa tudo
por playlists.


-O script "calcula_correlacao_atributos_x_popularidade.R" calcula a correlação de todos os atributos obtidos com a popularidade
(atributo x popularidade) criando um arquivo de saida com esses dados e também vários gráficos plot xy.
