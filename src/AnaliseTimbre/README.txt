- O primeiro script que deve ser executado é o timbresPlaylistMatching.R,
pois este prepara o arquivo com as músicas que têm timbre e estão contidas
na coleta do echonest.

- Depois executa o scrit CalculaSimilaridadeTimbre.py. Neste é efetuado efetivamente
o cálculo das similaridades entre os timbres das músicas par-a-par de cada playlist.

- Por fim, executa-se o script analisaTimbres, que cria uma tabela de estatisticas
referentes aos timbres.
